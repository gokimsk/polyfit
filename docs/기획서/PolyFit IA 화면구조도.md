**버전**: v1.0

**작성일**: 2025.08.02

## 🏗️ 사이트 구조

`PolyFit
├── 📱 스플래시 화면
│   └── 로딩 → 메인 페이지 자동 이동
│
├── 🏠 메인 페이지 (/)
│   ├── 헤더
│   │   ├── PolyFit 로고
│   │   └── 신청현황 버튼 (α기능)
│   ├── 메인 콘텐츠
│   │   ├── "요즘 이런 상황이신가요?" 헤딩
│   │   ├── 6개 상황카드 (2x3 그리드)
│   │   │   ├── 자취 시작
│   │   │   ├── 구직 중  
│   │   │   ├── 퇴사 후
│   │   │   ├── 육아 준비
│   │   │   ├── 연말정산
│   │   │   └── 결혼 준비
│   │   └── "선택한 상황으로 정책 찾기" 버튼
│   └── 푸터
│       └── "화면을 터치하여 건너뛰기" (선택사항)
│
├── 📋 정책 리스트 페이지 (/policies)
│   ├── 헤더
│   │   ├── 뒤로가기 버튼
│   │   ├── "맞춤 정책 (X개)" 제목
│   │   └── 실시간 버튼 (업데이트 상태)
│   ├── 선택된 상황 표시
│   │   └── 상황 태그들 (예: 육아 준비 + 퇴사 후 + 연말정산)
│   ├── 필터 영역
│   │   ├── 정렬 옵션
│   │   │   ├── 추천순 (기본)
│   │   │   ├── 마감임박순
│   │   │   └── 최신순
│   │   └── 카테고리 필터
│   │       ├── 주거지원
│   │       ├── 취업/교육  
│   │       ├── 생활/복지
│   │       ├── 금융/창업
│   │       └── 의료지원
│   └── 정책카드 리스트
│       └── 각 정책카드
│           ├── 상황 매칭 표시 (⭐ XX% 일치)
│           ├── 정책 제목
│           ├── 한줄 요약
│           ├── 실시 기관
│           ├── 대상 | 지원 | 기간 (테이블 형태)
│           └── 관련 태그들
│
├── 📄 정책 상세 페이지 (/policies/[id])
│   ├── 헤더
│   │   ├── 뒤로가기 ("목록")
│   │   └── 공유 버튼
│   ├── 메인 정보 카드
│   │   ├── 상황 아이콘 + 매칭 표시
│   │   ├── 정책 제목
│   │   └── 상세 설명
│   ├── 기본 정보 섹션
│   │   ├── 👥 지원대상
│   │   ├── ⭐ 지원내용  
│   │   └── 📅 신청기간
│   ├── 신청 프로세스 섹션
│   │   ├── 1️⃣ 온라인 신청
│   │   ├── 2️⃣ 서류 검토  
│   │   ├── 3️⃣ 현장 확인
│   │   ├── 4️⃣ 지원 결정
│   │   └── 5️⃣ 지원금 지급
│   ├── 상세 정보 아코디언
│   │   ├── 🟢 신청 자격 요건 (접기/펼치기)
│   │   ├── 📋 필요 서류 (접기/펼치기)  
│   │   └── ⚠️ 신청 시 주의사항 (접기/펼치기)
│   ├── α 기능 섹션
│   │   ├── 💡 활용 팁
│   │   ├── 👥 이용자 현황 ("이런 분들이 많이 찾아요")
│   │   └── ⭐ 이용자 후기
│   ├── 관련 정책 추천
│   │   └── 유사 정책 3-5개 카드
│   └── 신청하기 버튼
│       └── 외부 사이트 이동 확인 팝업
│
└── 🔖 α 기능들 (시간 여유 시)
    ├── 북마크 페이지 (/bookmarks)
    │   ├── 헤더 (뒤로가기 + 제목)
    │   └── 저장된 정책 리스트
    └── 신청현황 페이지 (/applications)  
        ├── 헤더 (뒤로가기 + 제목)
        └── 신청한 정책 현황 (localStorage 기반)`
